## アプリ作成手順
```bash
# nvmインストール
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/master/install.sh | bash

# nvmの確認　"nvm"が返ってくる
command -v nvm

# node.js LTS インストール
nvm ls
nvm install --lts
nvm install node

node --version
npm --version

# vueのインストール
npm install -g @vue/cli
vue --version

# アプリの作成
vue create aiart-app
cd aiart-app
npm run serve
```
